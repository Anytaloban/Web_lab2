# Web_lab2
Виконала Лобань Анна, КА-31.
За структурою в проєкті наявні 4 основних сторінки. Всі сторінки за своєю побудовою скдаються з шапки header, основної частини main та нижньої частини footer. Header містить картинку з логотипом продукту, рядок пошуку з іконкою лупи, та ненумерований список посилань на сторінки Головну, Про нас та Контакти, тобто меню навігації, яке реалізровано в окремому блоці nav.
Нижня частина сторінок сайту містить зображенн логотипу з підписом, а також список посилань (без конкретного значення) для підзаголовку Довідка. 
Сегмент main для сторінки Про нас є доволі простим. Він являє собою просто підзаголовок та абзац послідовного тексту з описом та призначенням даного сайту.
Сегмент main для сторінки Контакти містить 2 блоки контактної інформаціЇ. Перший відповідльний за соц мережі та складається з рядків з текстом та гіперпосиланнями. Другий блок відповдальний за бізнес-комунікацію та має подібну структуру до перереднього з єдиною відмінністю наявності модифікацій гіперпосилань. Значення цих рядків одразу дозволяють написати листа на вказану адресу або зателефонувати.
Сегмент main для сторінки Головна є найблільшим, адже складається аж з 4 чотирьох умовних частин. Перша являє собою стилістичне зображення, заголовок та рядок пошуку. Наступна частина містить підзаголовок та список зображень логотипів видавництв, які одночасно є гіперпосиланнями (без конкретного URL) на неналаштоване сортування. Наступний блок це Бестселери, тобто певний ненумерований список товарів. Кожен об'єкт списку має іконку сердечка (для списку обраного), підзаголовок з назвою книги, ціни, та посилання на сторінку з Деталями продукту. Останній блок це форма для реєстрації на сайті.
Сегмент main сторінки Деталі продукту ділиться на умовні 3 секції. Перша - інформація про товар (зображення, назва як заголовок, ціна та можливість відвідати сайт видавця за книпкою "На сайт видавництва" або доадти товар в Обране). Друга секція є ненумерованим списком книгарень, які мають вказаний товар. Серед об'єктів зображення логотип продавця, ціна за товар в цій книгарні та гіперпосилання на сайт для купівлі (наразі без конкретного значення). Остання секція є сукупністю заголовків та тексту для опису характеристик продукту.
